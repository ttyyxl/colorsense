## 二、项目基础信息

项目名称：ColorSense / ColorInsight。

一句话定位：基于 AI 四季色彩识别的个人色彩诊断与穿搭灵感 Web App，帮助用户通过照片理解自己的色彩类型，并获得更实用的风格建议。

项目灵感来源：团队在网上刷到韩国色彩测试相关内容，发现现代用户对人格测试、风格测试、色彩测试、MBTI 式自我认知工具有很高兴趣。同时，很多用户不只想知道“我是什么类型”，还希望得到每日穿搭、妆容、色卡、风格表达等更实用的参考。因此项目希望把“色彩测试 + AI 模型识别 + 风格推荐 + 网站产品化”结合起来。

当前项目已实现的主要能力：

- Firebase 邮箱注册、邮箱验证、邮箱登录、Google 登录、退出登录。
- 个人形象问卷与个人形象档案。
- 上传正面人像照片，进行清晰人脸检测与四季色型模型诊断。
- 结果页展示季型、置信度、四类分数、推荐色卡、风格关键词、避免色和 AI 建议。
- 历史记录页展示诊断记录和穿搭日记，支持批量操作和 PNG ZIP 导出。
- OOTD / 穿搭灵感功能：结合最近季型、个人档案、天气、场景和心情生成穿搭建议。
- 设置页支持修改密码和退出登录。

核心技术栈：

- 前端：Next.js 14、React 18、TypeScript、Tailwind CSS 4、Framer Motion、lucide-react。
- 认证与数据：Firebase Authentication、Cloud Firestore、Firebase Admin SDK。
- 后端推理：FastAPI、PyTorch、torchvision、MediaPipe、OpenCV、Pillow、NumPy、scikit-learn。
- AI 文案/建议：项目中包含 Gemini、OpenAI、Anthropic、豆包相关服务与 prompt，当前诊断流程中会生成豆包风格建议。
- 结果导出：html2canvas / html2canvas-pro、JSZip。
- 部署路线：Vercel 承载 Next.js 前端与 API Routes，Render 承载 FastAPI 推理服务，Firebase Spark 承载 Auth 与 Firestore。

### 第 6 页：模型训练流程与 AI 技术

核心信息：项目使用深度学习、CNN、迁移学习、颜色特征融合、数据增强和 EarlyStopping 等技术。

建议流程：

2. 意大利语标签映射：primavera / estate / autunno / inverno → spring / summer / autumn / winter。
3. OpenCV / MediaPipe 人脸检测与裁剪。

10. Macro F1：作为重要评估指标，避免只看 accuracy。

### 第 7 页：模型优化过程：从简单 CNN 到 EfficientNet-B0

核心信息：要重点讲“摸索过程”，不是只给最终结果。

建议版式：

- 时间线或阶梯式演进图，三阶段递进。

阶段 1：`v1_clean_cnn`

- 作用：作为 baseline，先验证数据读取、标签映射、训练评估、导出流程是否能跑通。
- 特点：简单 CNN，不使用预训练，不融合颜色特征。
- 局限：特征表达能力有限，对复杂人脸、光照和个体差异的适应能力不足。

阶段 2：`resnet18_color`

- 作用：开始引入预训练模型和颜色特征。
- 为什么改：单纯 CNN 学到的视觉特征不够稳定，迁移学习可以利用 ImageNet 预训练特征，提高小数据集下的泛化能力。
- 颜色特征价值：四季色彩判断高度依赖 Lab / HSV 等颜色信息，加入显式颜色统计特征可以补足 CNN 对颜色分布的表达。

阶段 3：`efficientnet_b0_color`

- 作用：最终候选模型。
- 为什么继续优化：EfficientNet-B0 在参数效率、特征表达能力和小模型部署之间更平衡。
- 结果：测试集综合表现最好，最终成为后端推理模型。

请在页面中突出一句结论：我们不是一开始就得到最终模型，而是先用简单模型验证流程，再逐步引入迁移学习和颜色特征，最后选择综合泛化能力更好的 EfficientNet-B0。

## 四、必须在 PPT 中体现的真实项目细节

请务必把以下项目事实准确写进 PPT：

- 训练数据中原始样本 4920 张，有效 face crop 4660 张。

  - `efficientnet_b0_color`：best_epoch 12，best_val_macro_f1 0.557834，test_loss 0.981513，test_accuracy 0.600858，test_macro_f1 0.599448。
- 最终选择 `efficientnet_b0_color` 的原因：测试准确率与测试 Macro F1 最高，综合泛化表现最好。
